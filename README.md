# Projeto_CENARIOS

<img width="800" height="800" alt="PROJETO CENARIOS (2)" src="https://github.com/user-attachments/assets/2c7d8c21-fec4-4657-b001-8357ebad1cb7" />

O Projeto teve como objetivo central projetar cenários de intensificação agropecuária e conservação de remanescentes, simulando o desmatamento futuro nos biomas Amazônia, Cerrado e Pantanal em função da expansão de pastagens e soja. Paralelamente, a análise de desmatamento evitado foi uma das metas para a compreensão das dinâmicas que impulsionam as transições do uso da terra. Essa é uma iniciativa entre o Laboratório de Processamento de Imagens e Geoprocessamento (LAPIG - UFG) e The Nature Conservancy.

<br> 

This repository contains the following:
- Data: Input spatial data
- Model: Dinamica EGO model and required sub-models
- Scripts: R, JavaScript, Python, Bash scripts
